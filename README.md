# D3 Status
D3 Status is a JRuby app that allows you to check the current status of Diablo 3 servers (US only)


##System Requirements 

* Java Runtime
* JRuby
* nokogiri gem



##How to run

```
git clone git://github.com/JohnRedmond/D3Status.git
cd D3Status
jruby app.rb
```
